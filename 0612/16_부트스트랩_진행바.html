13_진행바

스티키형 막대, 애니메이션 배경 및 텍스트 레이블을 지원하는 bootstrap 사용자 정의의 프로그레스 바

프로그레스 컴포넌트는 2개의 html요소와 너비를 설정하기 위한 몇 가지 css와 속성들로 빌드된다.

1. html5에서의 프로그레스바

progress

2. bootstrap에서의 프로그레스바

progress 태그요소는 사용하지 않으므로, 프로그레스 바를 쌓아 올리거나 애니메이션화 하거나 텍스트 레벨을 배치할 수 있다.

  • 프로그레스 바의 최대 너비를 나타내기 위해 progress를 사용한다.
  • 진행상황을 나타내기 위해 안쪽에 class="progress-bar"를 사용한다.
  • class="progress-bar"는 폭을 설정하기 위해 인라인 스타일, 유틸리티 클래스 또는 사용자 정의 css를 필요로 한다.
  • class="progress-bar"는 액세스 할 수 있도록 몇개의 role과 aria 속성을 필요로 하며 접근성이 있는 이름(aria-label, aria-labeledby 등)을 포함한다.

3. 프로그레스 바 기본 설정

class="progress"를 쓰면 100% 크기의 프로그레스바가 생성된다. (아래는 보기 편하게 80%로 수정됨)

PHP : 0%
HTML5 : 25%
CSS3 : 50%
JAVASCRIPT : 75%
JQUERY : 100%

4. 프로그레스 바 색상 변경

배경색 변경시 class="progress-bar" 안에 "bg-type(ex_success)" 클래스 서식을 추가한다.

글자색 변경시 class="progress-bar" 안에 "text-type(ex_success)" 클래스 서식을 추가한다.

6. 프로그레스 바에 스프라이트 무늬와 애니메이션 적용하기

스프라이트 무늬 적용시 class="progress-bar" 안에 "progress-bar-striped" 클래스 서식을 추가한다.

스프라이트 무늬 적용 후 애니메이션 적용시 class="progress-bar" 안에 "progress-bar-animated" 클래스 서식을 추가한다.
