05_문단_텍스트관련

부트스트랩에서는 기본 글자크기가 16px, 행높이(줄간격)은 1.428값이 기본값이다.

위 내용은 문단태그(p)요소에 자동으로 적용되며, 하단여백(margin)은 16px이 기본값이다.

제목요소의 마진값은 기본 8px로 설정(아래)

small : 글자를 작게 설정함(14px)
mark : 글자에 하이라이트 효과를 적용함(16px)
abbr : 약어나 줄임말을 사용하고자 할 때 사용함(16px)
blockquote : 다른 사람의 말이나 글을 대신하여 작성할 때 사용함(16px)
blockquote class 'blockquote-footer' : 다른 사람의 말이나 글을 대신하여 작성할 때 사용함(글자크기 14px, 색상 #6c757d)

문단관련 서식

class 'lead' : 단락을 눈에 띄게 하기(글자크기 20px)

class 'small' : 상위문자 크기의 85% = 14px로 문자크기를 작게 설정함

class 'text-start' : 문단정렬(왼쪽 정렬)

class 'text-end' : 문단정렬(오른쪽 정렬)

class 'text-center' : 문단정렬(가운데 정렬)

class 'text-justify' : 양쪽 배분 정렬

class 'text-nowrap' : 문단 끝에서 글자가 다음 줄로 넘어가지 않게 설정함

class 'text-uppercase' : 영문자를 무조건 대문자로 출력

class 'text-lowercase' : 영문자를 무조건 소문자로 출력

class 'text-capitalize' : 영문자의 첫글자만 대문자로 출력

목록태그

  • class 'list-unstyled' : 목록 기호를 숨긴다.
  • 목록태그 가로배치 방법 : inline, inline-block, float:left, flex, grid
  • 부트스트랩에서는 class 'list-inline-item'으로 가로배치 할 수 있다.
