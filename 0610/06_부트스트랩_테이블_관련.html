검색이나 버튼
  • table 태그에 class 'table' : 테이블 기본 서식 적용 (가로로 연회색 줄)
  • table 태그에 class 'table-striped' : 홀수 행에 다른 색상을 적용 (연회색 배경)
  • table 태그에 class 'table-bordered' : 테이블 기본 서식 적용 (세로로 연회색 줄)
  • table 태그에 class 'table-hover' : 호버시 행에 배경 색상을 적용 (연회색 배경)
  • thead 태그에 class 'table-dark' : thead(티헤드)에 배경색, 글자색 설정 (블랙/글자컬러 화이트)
  • table 태그에 class 'talbe-responsive' : 화면의 너비가 768px 미만일 경우 가로로 스크롤 설정
  • tr 태그에 class 'table-success' : tr(행)에 배경색 설정 (그린)
  • tr 태그에 class 'table-success' : tr(행)에 배경색 설정 (그린)
  • tr 태그에 class 'table-info' : tr(행)에 배경색 설정 (하늘)
  • tr 태그에 class 'table-primary' : tr(행)에 배경색 설정 (파랑)
