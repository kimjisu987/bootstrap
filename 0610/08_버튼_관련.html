08_버튼 관련

버튼은 button태그로 사용하여 클래스명을 적용한다.

  • 어둡게 : class="btn btn-dark"
  • 밝게 : class="btn btn-light"
  • 블루 : class="btn btn-primary"
  • 그레이 : class="btn btn-secondary"
  • 그린 : class="btn btn-success"
  • 하늘 : class="btn btn-info"
  • 노랑 : class="btn btn-warning"
  • 빨강 : class="btn btn-danger"
  • 일반링크 : class="btn btn-link"

button 태그 / input 태그[type="button"] / a 태그link

버튼 테두리만 적용

버튼 크기지정하기 class="btn-lg"(20px), class="btn-sm"(14px)

버튼 비활성화 disabled

a 비활성화

a태그의 경우 class="disabled"를 넣어줘야 함

버튼블럭 class="d-grid gap-2"

50%의 가로폭으로 지정시 class="col-6"

수평으로 중앙 배치시 class="mx-auto"

버튼그룹 class="btn-group"
