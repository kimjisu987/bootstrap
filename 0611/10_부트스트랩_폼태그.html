10_폼태그

다양한 폼 사용을 위한 폼 컨트롤 스타일, 레이아웃 옵션, 사용자 정의 컴포넌트의 예시와 사용 가이드입니다.

폼태그 사용시 form, fieldset, legend 태그 사용

폼태그 그룹제목

1. TEXT 박스

2. PASSWORD 박스

3. 폼 양식 크기 조정

폼 입력 양식의 크기 조정은 class="form-control-lg" 또는 "form-control-sm"으로 크기를 조정할 수 있다.

4. 설명문구 추가

class="form-text" block요소와 간격을 쉽게 맞출 수 있도록 상단 여백이 추가된다. (설명문구)

아이디는 영문 소문자, 숫자만 입력해주세요.

5. 버튼 비활성화 - disabled

disabled 불리언 속성을 추가하면 비활성화 외형이 되며, 포인터 이벤트가 삭제되면서 포커싱되지 않습니다.

6. 읽기 전용 - readonly

readonly 불리언 속성을 추가함으로써 입력값의 변경을 막을 수 있습니다. disabled와 달리 readonly 입력값은 계속 포커싱 및 선택이 가능합니다.

7. 웹 접근성 - aria-label

웹 접근성을 높이기 위해서는 aria-label="대체 텍스트"를 작성해줘야 한다. aria-label은 모든 html 태그에 사용할 수 있다.

8. 라디오버튼

라디오버튼은 input 태그 name=""은 동일하게, id=""는 다르게 작성해야 한다.

label 태그에는 class="form-check-label", input 태그에는 class="form-check-input"을 적용한다.

부트스트랩 디자인

9. 체크박스

라디오버튼과 마찬가지로 label 태그에는 class="form-check-label", input 태그에는 class="form-check-input"을 적용한다.

10. 선택목록 - select, option

select 태그에 class="form-select" 서식을 적용한다.

(1) 크기 조정시 class="form-select-lg" 또는 "form-select-sm" 서식을 추가로 적용한다.


(2) 다중 선택 선택목록을 만들 때는 select 태그에 multiple 서식을 추가한다.
(3) 옵션의 높이는 size="줄수"로 조정이 가능하다.
(4) 하나의 그룹으로 만들 때는 div 부모 박스를 만들어서 class="input-group" 서식을 추가한다.
@

11. 다중텍스트 박스

(1) class="form-control"일 때는 가로 100%, "form-control-lg", "form-control-sm"을 적용할 수 있다.


(2) 높이를 적용하고 싶으면 css서식으로 조절이 가능하다. (style="height:100px;")
